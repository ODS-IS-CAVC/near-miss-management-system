string <uuid> = 36 characters

string [ 0 .. 4096 ] characters
Example: name=浜松

string [ 0 .. 4096 ] characters
Example: category=cat12-

summary
string [ 0 .. 4096 ] characters
Example: summary=追い越し

from
string <date-time>
Example: from=2024-07-10T09:37:58.372Z

発生時刻フィルター。下限値（包含）

to
string <date-time>
Example: to=2024-07-11T09:37:58.372Z

発生時刻フィルター。上限値（排他）

lat0
number <double>
Example: lat0=35.66104767541138

発生地点フィルター。下限緯度（包含）

lon0
number <double>
Example: lon0=139.70993603470313

number <double>
